### ЁЯУ▒ Fully Responsive тАФ Mobile + Desktop

### ЁЯУж Module Cards

Each phase has multiple learning modules. Every card shows:
- Module title and depth tag (`Deep` / `Core` / `Practical`)
- Total concept count
- Preview of the first 3 concept names with live tick status
- Progress bar showing completion

Tap any card to open the full concept checklist for that module.

---

### тЬЕ Concept Checklist

Inside each module, every concept has:
- A **name** тАФ the specific topic to learn
- A **detail line** тАФ exactly what to study, which tools, and why it matters on the job
- A **circular checkbox** тАФ tap to mark as learned, fills with the phase colour

Example from *Linux & Bash Mastery*:
```
тЧЛ  SSH & Remote Work
   ssh keys, scp, rsync, tmux/screen, port forwarding тАФ
   working with remote GPU servers and cloud VMs daily
```
Tap тЖТ turns green with тЬУ, saved to `localStorage` instantly.

---

### ЁЯТ╛ Persistent Progress

Every checkbox is saved to `localStorage` the moment you tap it:

- тЬЕ Close tab тЖТ reopen тЖТ everything exactly where you left it
- тЬЕ Works fully offline after first load
- тЬЕ Last active phase tab is also restored on reopen
- тЬЕ `тЬУ Saved` flash confirms each save on desktop
- тЪая╕П Progress is per-browser тАФ Chrome and Firefox store separately

---

### ЁЯУК Live Progress Tracking тАФ 3 Levels

All bars update instantly on every tap тАФ zero page reload.

## ЁЯЫая╕П Tech Stack

| Layer | Technology | Why |
|-------|-----------|-----|
| Language | Vanilla JavaScript (ES6+) | No build step, instant first render on any device |
| Styling | CSS3 with real `@media` queries | Proper responsive design, no JS breakpoint hacks |
| Rendering | Direct DOM manipulation | Zero framework overhead |
| Persistence | Browser `localStorage` | Offline-capable, zero-server progress saving |
| Hosting | GitHub Pages | Free, CDN-backed, HTTPS |

**No React. No Babel. No npm. No node_modules. No build pipeline.**
One `index.html` тАФ open it anywhere and it works instantly.

> The previous v1.x used React 18 + Babel Standalone (~1.2MB runtime that had to compile JSX on the device before anything rendered). v2.0 ships zero runtime dependencies тАФ the browser gets pure HTML, CSS, and JS as intended.
